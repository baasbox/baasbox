This is the list of the roles managed by BaasBox. Some of them cannot be edited.
All roles give the same permissions of the "registered" one.
You can define new roles and assign users to them. This can be useful if you want to give access to some documents only to some specific role by setting up the specific grants.
You can add new roles by clicking on the 'New Role' button on the bottom of the list.
